# Amrop və Demirbank Təqdimatı

Bu tətbiq Amrop və Demirbank üçün birgə strateji əməkdaşlıq təqdimatıdır. İşıqlı, təmiz və müasir dizayn prinsipi ilə Flutter istifadə edilərək yaradılmışdır.

## Xüsusiyyətlər
* **Giriş və Loqolar:** İlk slayd hər iki şirkətin – DemirBank və Amrop-un loqotipləri ilə tərtib edilib.
* **Qısa və Məzmunlu:** Slaydlar asan oxunan və təmiz ağ fonda (DemirBank-ın qırmızı vurğuları ilə) dizayn edilib.
* **Kompensasiya (Compensation):** Şirkət əməkdaşları üçün nəzərdə tutulmuş güzəşt və kompensasiyaları aydın göstərən xüsusi slayd.
* **Komandamız:** Dəyanət, Orxan Məmmədov və Səbinənin daxil olduğu peşəkar komanda profilləri.
* **Əlaqə:** Təqdimatın sonuncu slaydında əlaqə məlumatları və birgə tərəfdaşlıq vurğusu yer alır.

## Texnologiyalar
* Flutter
* Dart

## Necə işə salmalı
Tətbiqi lokal mühitinizdə işə salmaq üçün aşağıdakı əmrləri icra edin:

```bash
flutter pub get
flutter run
```
